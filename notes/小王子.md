# Python 列表基础知识

## 一、什么是列表
列表是 Python 最常用的可变序列数据类型，用于存储一组有序的元素：
```python
# 定义列表
nums = [1, 2, 3, 4, 5]
names = ["张三", "李四", "王五"]
mix = [10, "hello", True, 3.14]
